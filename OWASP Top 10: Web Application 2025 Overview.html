What is OWASP?

The Open Worldwide Application Security Project (OWASP) is a nonprofit foundation dedicated to improving software security worldwide through community-driven open-source initiatives.

  • Community-led open source projects
  • 250+ local chapters worldwide
  • Tens of thousands of members
  • Industry-leading guidance, training, and conferences

What is the OWASP Top 10?

The OWASP Top 10 is a standard awareness document identifying the most critical security risks facing modern applications.

  • Web Applications
  • Mobile Applications
  • APIs
  • Desktop Applications
  • Large Language Model Applications

OWASP Top 10 (2025)

  • A01: Broken Access Control
  • A02: Security Misconfiguration
  • A03: Software Supply Chain Failures
  • A04: Cryptographic Failures
  • A05: Injection
  • A06: Insecure Design
  • A07: Authentication Failures
  • A08: Software and Data Integrity Failures
  • A09: Security Logging and Alerting Failures
  • A10: Mishandling of Exceptional Conditions

Changes from 2021 OWASP Top 10

Changes Overview

  • Supply chain and third-party components are recognized as primary risk factors
  • Misconfiguration reframed as a design and default settings problem
  • Business logic abuse elevated due to real-world attack prevalence
  • Improper error handling now recognized as a root cause of information disclosure and business logic abuse

What This Changes for Developers

  • Development environments and CI/CD pipelines are now part of the attack surface
  • Design and architecture reviews matter as much as vulnerability scanning
  • Third-party dependencies require continuous inventory and monitoring
  • Business logic testing cannot be fully automated
  • Identity, authorization, and trust boundaries must be explicit and tested

OWASP Juice Shop Demos

OWASP Juice Shop is a modern, intentionally vulnerable web application designed for security training, awareness demos, CTFs, and testing security tools. It includes vulnerabilities covering the entire OWASP Top 10 as well as many additional real-world security flaws, making it an ideal hands-on learning platform.
